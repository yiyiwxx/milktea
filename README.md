# 今日奶茶

不是你在选奶茶，是你的心情在选。

选三个词——晚风、温柔、星辰——然后让一杯茶替你说话。

## 它到底特别在哪？

### 不是菜单，是读心术
市面上的奶茶推荐只是按品类筛选，但在这里你需要**选三个词**。你的心情会被 84 个诗意标签捕捉（晚风、山海、缱绻、初雪……），然后从 86 款饮品中配出最合拍的那一杯。

### 每杯茶都有两副面孔
每款饮品配了两段完全不同的推荐语——一段走心，一段搞钱。每次刷新结果随机出现，所以同一杯茶你可以喝出不同味道。

### 有结果的饮品推荐
你喝到的不是冰冷的匹配，而是一整套体验：饮品名 → 品牌徽章 → 甜度/温度/加料标签 → 推荐理由 → 今日幸运签。甚至连右下角的装饰图标都会根据品类自动切换（果茶、奶茶、纯茶、咖啡、酸奶奶昔各有专属卡通图）。

### 用手账的方式做 UI
每一步都有线条小狗陪你：选词时词卡上有小狗图案，选中后顶部展示栏也有小狗，结果页每个标签里都藏着云朵和狗爪印。开始页也有随机小狗迎接你。

### 覆盖 15 个品牌 86 款饮品
霸王茶姬、喜茶、蜜雪冰城、奈雪的茶、茶百道、古茗、瑞幸咖啡、沪上阿姨、CoCo 都可、茶颜悦色、一点点、益禾堂、茉莉奶白、书亦烧仙草、爷爷不泡茶——主流品牌全覆盖。

## 怎么用

用浏览器打开 `每日奶茶推荐.html` 即可。

```
点击「开始今日奶茶」→ 第 1 轮选词 → 第 2 轮选词 → 第 3 轮选词
→ 看看结果 → 匹配的饮品 + 推荐理由 + 幸运签
→ 不满意？「再来一杯」重新选
```

你的推荐记录会自动保存在浏览器中。

## 关于数据

饮品信息和推荐语均为原创整理。如果喝到了想喝的，那就是匹配系统在认真工作；如果喝到了离谱的，那就是那 1% 的彩蛋。

> 糖分摄入不宜贪多，今天喝完，要记得多喝水哦。
